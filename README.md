# AutoML Algorithm Recommendation System Using Meta-Learning

A meta-learning system that analyzes any CSV dataset and recommends the best ML algorithms with expected performance estimates.

## Project Structure

```
automl-algorithm-recommender/
├── data/
│   ├── raw/
│   │   └── meta_dataset.csv           # Raw meta-learning training data
│   └── processed/
│       └── clean_meta_dataset.csv     # Cleaned dataset (generated by EDA.py)
├── models/
│   ├── meta_model.pkl                 # Trained meta-learning model
│   └── label_encoder.pkl              # Label encoder for algorithm classes
├── results/
│   └── model_metrics.json             # Evaluation metrics for all trained models
├── src/
│   ├── EDA.py                         # Data cleaning and EDA pipeline
│   ├── model_pipeline.py              # Multi-model training with GridSearchCV
│   ├── train_model.py                 # Training entry point
│   ├── predictor.py                   # Algorithm recommendation predictor
│   ├── performance_predictor.py       # Expected accuracy estimator
│   ├── meta_feature_extractor.py      # Dataset meta-feature extraction
│   └── dataset_analyzer.py            # Dataset analysis and complexity scoring
├── app/
│   └── streamlit_app.py               # Interactive web interface
├── notebooks/
│   └── eda_analysis.ipynb             # EDA analysis notebook
├── requirements.txt
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### Step 1: Run EDA and Data Cleaning
```bash
python src/EDA.py
```

### Step 2: Train the Model
```bash
python src/train_model.py
```

### Step 3: Launch the App
```bash
streamlit run app/streamlit_app.py
```

The app opens at `http://localhost:8501`.

## Pipeline Overview

```
Raw Dataset (meta_dataset.csv)
        ↓
    EDA.py  →  clean_meta_dataset.csv
        ↓
 model_pipeline.py  →  GridSearchCV (LR, RF, XGBoost)
        ↓
  Best Model  →  meta_model.pkl + label_encoder.pkl
        ↓
  streamlit_app.py  →  Upload CSV → Recommendations
```

## Meta-Features Extracted

| Feature | Description |
|---|---|
| n_samples | Number of rows |
| n_features | Number of features |
| imbalance_ratio | Class imbalance ratio |
| avg_corr | Average feature correlation |
| class_sep | Class separability score |
| missing_value_ratio | Proportion of missing values |
| numeric_feature_ratio | Proportion of numeric features |
| categorical_feature_ratio | Proportion of categorical features |

## Models Trained

| Model | Tuning |
|---|---|
| Logistic Regression | C, penalty, solver |
| Random Forest | n_estimators, max_depth, min_samples |
| XGBoost | n_estimators, max_depth, learning_rate, subsample |
